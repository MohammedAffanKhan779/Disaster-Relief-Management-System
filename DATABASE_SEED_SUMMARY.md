## 🔄 Database Reset Instructions

If you want to reseed with fresh data:

1. **Stop the application** (if running)
2. **Drop the database:**
   ```sql
   DROP DATABASE disaster_management;
   ```
3. **Restart the application** - Database will be recreated and reseeded automatically

### How DataLoader Works
- ✅ Runs automatically on application startup
- ✅ Checks if data exists (`user` table record count)
- ✅ If data exists, skips seeding (prevents duplicates)
- ✅ Delete all users to trigger fresh seed on next restart
